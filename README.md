# 🌦️ Weather App

A simple and responsive Weather Application that provides real-time weather updates for any city using the OpenWeatherMap API.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Displays current temperature (°C)
- 💧 Shows humidity level
- 🌬️ Displays wind speed
- 🌤️ Dynamic weather icons
- ⚡ Real-time API data fetching

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- OpenWeatherMap API
